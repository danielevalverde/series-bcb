Séries BCB

URL básica

#server<metodo>/<parametro>/<resposta>

Esta API é destinada para exibir valor de uma determinada série pública.

  • ultimoValor
    • parâmetro -> formato: numeroSerie
      • numeroSerie: inteiro
  • valores
    • parâmetro -> formato: numeroSerie|dataInicio|dataFim
      • numeroSerie: inteiro
      • dataInicio: data (ddMMaaaa)
      • dataFim: data (ddMMaaaa)
  • json -> padrão
  • xml
